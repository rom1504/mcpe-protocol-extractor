# mcpe-protocol-extractor

Extract the protocol from pocket mine

## Usage

* `git clone -b mcpe-0.12 https://github.com/PocketMine/PocketMine-MP.git /tmp/pocketmine`
* `node protocol-extractor /tmp/pocketmine`
* `cat protocol.json`
